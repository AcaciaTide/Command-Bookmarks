# Command Bookmarks
A client-side Fabric mod that allows you to bookmark frequently used commands and chat messages for one-click execution via a dedicated GUI.
Stop typing the same long commands over and over again!

## Features
- **Easy Access**: Press the `V` key (customizable in controls) to open the bookmark list anytime.
- **Save & Execute**: Register any commands (e.g., `/time set day`) or regular chat messages and execute them with a single click `▶`.
- **Edit & Delete**: Easily modify existing bookmarks `✏` or remove them `🗑` when no longer needed.
- **Reordering**: Move your most important bookmarks to the top using the `↑` and `↓` buttons.
- **Smart Tooltips**: Hover over long commands in the list to view their full text.
- **Client-Side Only**: Works everywhere. You don't need to install this mod on the server to use it in multiplayer!

## Supported Versions
- **Minecraft**: 1.21.11 (Fabric)
- **Requirements**: Fabric Loader & Fabric API

## How to Use
1. Press `V` (default) to open the Command Bookmarks GUI.
   - **Changing the Keybind:** You can change this key at any time by going to Minecraft's **Options** -> **Controls** -> **Key Binds** -> **Miscellaneous** -> **Open Command Bookmarks**.
2. Click **[Add New]**.
3. (Optional) Enter a label to easily identify what the command does.
4. (Required) Enter your command (starting with `/`) or a chat message.
5. Click **[Save]**.
6. The next time you need it, simply open the GUI again and press the `▶` execute button next to your bookmark!
